# Stonebird Group Website v2

Static website for Stonebird Group.

## Deploy to GitHub Pages

1. Unzip this package.
2. Upload the contents of this folder to the root of the GitHub repository, not the folder itself.
3. Keep the `CNAME` file in the repo root. It should contain `stonebirdgroup.com`.
4. Commit to `main`.
5. Wait for GitHub Actions / Pages deployment to complete.
6. Visit `https://stonebirdgroup.com/?v=2` to bypass cache.

## Core files

- `index.html`
- `styles.css`
- `script.js`
- `assets/`
- `CNAME`

## Brand assets

The processed logo assets are in `/assets`. Original supplied logo PNGs are preserved in `/assets/source`.
