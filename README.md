# Stonebird Group website

A static one-page website for Stonebird Group.

## Files

- `index.html`: the site structure and copy.
- `styles.css`: visual design and responsive layout.
- `script.js`: subtle reveal animation.
- `assets/stonebird-mark.svg`: black mark for light backgrounds.
- `assets/stonebird-mark-dark.svg`: black circular mark with white linework.
- `assets/favicon.svg` and `assets/favicon.png`: browser and mobile icons.
- `assets/og-image.png`: social preview image.
- `robots.txt` and `sitemap.xml`: basic search engine files.

## Before publishing

1. Replace `hello@stonebirdgroup.com` in `index.html` if you use another inbox.
2. Replace or refine the SVG logo once the final production vector is approved.
3. Upload the full folder to your host, or drag the folder into Netlify / Vercel.
4. Point `stonebirdgroup.com` DNS to the host.

## Design notes

The site uses system fonts only. No external libraries, font files, trackers, or dependencies are included.
